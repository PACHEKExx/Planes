# Nuestros días

Generador de invitaciones para parejas, hecho con HTML, CSS y JavaScript. No usa cuentas, servidor ni base de datos.

## Probarla

Abre `index.html` o sirve la carpeta con Live Server. Completa el formulario y la aplicación generará una imagen PNG que puedes compartir desde el celular o descargar.

El botón **Compartir imagen** usa el menú nativo del teléfono. El botón **Abrir WhatsApp** prepara el texto; por seguridad, los navegadores de escritorio no permiten adjuntar automáticamente un archivo a WhatsApp, así que en ese caso descarga la imagen y adjúntala manualmente.

## Publicar en Vercel

Importa la carpeta desde GitHub en Vercel. No necesita comando de compilación: el proyecto es completamente estático.
